# timein
Give the time in a city from the cli

## Installation
```shell
$ go get -u github.com/Oxyriser/timein
```

## Usage
```shell
$ timein <city>
```

## Example
```shell
$ timein los angeles
20:30:29
$ timein new york
23:31:12
```

## Notes
Only works with cities registered in the IANA Time Zone database
